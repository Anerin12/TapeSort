# **TapeSort**  

#### TapeSort - это проект созданный в рамках выполнения тестового задания для поступления на стажировку в компанию **YADRO** команду *Tatlin.Unified DataPath*.

##### После сборки и запуска данный репозиторий представляет из себя консольное приложение, обеспечивающее сортировку данных из устройств, которые эмулируют устройства типа лента, в условиях ограничений по выгрузке чисел в оперативную память.

## 1. Сборка и запуск консольного приложения
##### Для сборки проекта в корневой директории проекта выполните данные команды 
```
mkdir build/ && cd build/ 
cmake ..
cmake --build .
```
##### Для запуска проекта из директории `build/` выполните:
```
./TapeSortApp <путь до входного файла> <путь до выходного файла (опционально)>

// Пример

./TapeSortApp ../input.txt ../output.txt
```

##### Для запуска тестов из директории `build/`, после сборки выполните:
```
./RunAllTests
``` 

## 2. Изменение конфигурации
##### Зайдите в файл `/config/config.txt`
```
rw_delay=0 // задержка на чтение/запись
rewind_delay=0 // задержка пролистывания ленты
move_delay=0 // задержка движения магнитной голоовки
ram_size=1500 // ограничение по оперативной памяти (в байтах)
```
##### Для настройки конфигурации просто измените значения в данном файле.

## 4. Ключевые решения
#### Tape:
##### Для корректной эмуляции типа лента на обычном файле, было принято решение читать из файла и записывать в файл char блоками по 12 байт. Это решение обусловлено тем что оно:
###### 1. Предоставляет данные в человеко-читаемом виде.
###### 2. Поддерживает перемещение по "ячейкам" ленты.
###### 3. Позволяет записывать integer ($2^{32}$).

#### TapeSort:
##### В данном классе я реализовал внешнюю сортировку слиянием. Промежуточные ленты 1-го слоя сортируются при помощи `std::sort()`.
##### В функции слияния для обеспечения скорости было принято решение использовать очередь с приоритетом для одновременного слияния нескольких лент. 

#### Logger:
###### Настроил систему логирования, реализовав класс Logger. 

#### generator:
###### При необходимости можно сгенерировать входную лентту вызвав функцию `generate_tape(int num, std::string file_name)` из `generator.hpp`. Где `num` -- колличество чисел в ленте, `file_name` -- имя файла для сохранения сгенерированной ленты.
